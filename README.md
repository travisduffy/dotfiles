# dotfiles
Configurations and setup scripts for my dev environment (Ubuntu-flavoured!)  
To use:  
  
```
sudo apt install git
cd ~
git clone https://github.com/travisduffy/dotfiles.git
cd dotfiles/scripts && source ENGAGE.sh
```
